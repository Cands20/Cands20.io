Tipos de Selectores

Selector de Tipo
Selecciona un tipo de elemento HTML y aplica el estilo a todos los elementos de ese tipo
Ejemplo:
p{text-align: justify;}
Selector descendente
Aplica el estilo a elementos que estan conectados dentro de otros elementos
Ejemplo:
p b{color: red;}
ul li b{color:green;}
Selector hijo
Similar al descendente pero mas restrictivo, aplica solamente el estilo a los elementos que sean hijos directos de otro elemento
Ejemplo:
p>b{color:blue;}
Selector por id
Un id es un atributo de un elemento HTML y se utiliza para aplicar un estilo de manera particular
Ejemplo:
p id="introduccion"
Este es el parrafo /p
# <= selector por id
#introduccion{ color: blue;}
Selector por clase
Aplicar estilo a un conjunto de elementos
p class="especial"
. <= selector por clase
.especial{color:red;}
Selector universal
Aplica el mismo estilo a todos los elementos de la pagina
